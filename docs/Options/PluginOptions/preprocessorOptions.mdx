# Preprocessor Options

Option: `preprocessorOptions`

Type: `Object` Default: `{}`


Here can be specified template engine options.
Each preprocessor has its own options, depend on using template engine.

The preprocessor specifically options you can find in the Guides for [Template Engines](/category/template-engines).
