# Installation

## Requirements

- [Node.js][node-url] >= 18
- [Webpack][webpack-url] >= 5.81.0 (recommend 5.96+ for supporting all features)

## Install

Install the plugin using your favorite package manager:

<Tabs groupId="tools">
  <TabItem value="npm" label="npm">
    ```bash
    npm install --save-dev html-bundler-webpack-plugin
    ```
  </TabItem>
  <TabItem value="yarn" label="yarn">
    ```bash
    yarn add --dev html-bundler-webpack-plugin
    ```
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    ```bash
    pnpm add --save-dev html-bundler-webpack-plugin
    ```
  </TabItem>
</Tabs>


## Known Webpack loaders/plugins

### Recommended loaders

| Resources | Packages                             | Notes                   |
| --------- | ------------------------------------ | ----------------------- |
| JS/TS     | [babel-loader][babel-loader-url]     | Optional                |
| JS/TS     | [swc-loader][swc-loader-url]         | Optional                |
| CSS       | [css-loader][css-loader-url]         | Required for CSS styles |
| CSS       | [postcss-loader][postcss-loader-url] | Optional                |
| SASS/SCSS | [sass][sass-url], [sass-loader][sass-loader-url]       | Required for SASS/SCSS  |

### Recommended optimization plugins
